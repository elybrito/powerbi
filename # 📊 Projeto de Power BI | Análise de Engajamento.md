Projeto desenvolvido com foco em portfólio para análise de dados.

Análise de Engajamento de Conteúdos Internos — Órion Tecnologia

Este projeto tem como objetivo analisar o engajamento dos colaboradores com conteúdos internos corporativos, utilizando o Power BI para transformar dados em insights estratégicos.

O dashboard foi desenvolvido com foco em responder perguntas de negócio como:

Os colaboradores estão consumindo os conteúdos disponibilizados?
Existe diferença de engajamento entre conteúdos novos e antigos?
Quais áreas apresentam maior ou menor adesão?
O nível hierárquico influencia o consumo de informação?

Principais Insights:

O relatório apresenta uma visão consolidada do comportamento dos usuários, destacando:

Taxa geral de leitura dos conteúdos disponibilizados
Comparativo entre conteúdos novos vs. antigos, evidenciando padrões de interesse
Análise por segmento organizacional (Comunicação, Financeiro, RH e TI)
Relação entre engajamento e hierarquia (Coordenador, Supervisor, Operador)
Impacto da senioridade (Novato vs. Veterano) no consumo de conteúdo

Abordagem Técnica:

O projeto foi estruturado com as seguintes etapas:

Tratamento e transformação de dados utilizando Power Query
Modelagem de dados com foco em performance e clareza analítica
Criação de medidas com DAX para cálculo de indicadores de engajamento
Desenvolvimento de um dashboard interativo e intuitivo

Diferencial do Projeto:

Para garantir a confidencialidade, os dados utilizados foram anonimizados e adaptados, mantendo a estrutura real de um ambiente corporativo.

O projeto simula um cenário real de negócio, demonstrando na prática:

Capacidade analítica
Organização de dados
Construção de indicadores relevantes
Comunicação visual eficiente
